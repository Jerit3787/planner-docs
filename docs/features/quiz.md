# Quiz & Tests

Track quizzes, tests, and Final Exams — with scores, status, and an optional linked focus/revision
trail.

## Regular quizzes/tests

Set a date and (optionally) a time, a topic link, and a status. Once the scheduled time passes, an
unfinished quiz automatically flips to **Completed** — you don't have to remember to close it out
yourself. Add a score once it's graded, and it feeds the Dashboard's score-trend chart.

## Final Exams

Toggle **Final Exam** on the same form to switch it into exam mode — this adds a seat/table number
field and a "Morning session" / "Evening session" quick-pick for the exam's start time (both configured
in [Settings](settings.md)), since final exam schedules are usually published in session blocks rather
than exact times. A Final Exam is otherwise the same underlying record as a regular quiz — topic links,
focus sessions, and score all carry over.

## Pomodoro focus sessions

Log a focus/study session against a quiz to build up a time-by-subject breakdown on the Dashboard. If
you have an active Pomodoro timer running when one of today's classes or quizzes is about to start, it
automatically pauses itself instead of silently running through it.
